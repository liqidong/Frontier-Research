# Vibe Coding

> Coined by Andrej Karpathy. A new paradigm of software development in the age of LLMs.

## 📝 Definition
Vibe Coding is a style of programming where the developer focuses on the "vibes," high-level descriptions, and feedback loops with AI, rather than writing every line of code manually. It treats LLMs as the primary implementers and the human as the product engineer and reviewer.

## 🚀 Key Characteristics
- **Exponential Speedup**: Coding tasks that took months now take hours or days.
- **Taste over Syntax**: Human "taste" and "product intuition" become the most valuable skills as implementation cost drops to near zero.
- **Roll and Re-roll**: Instead of surgical debugging, developers often discard and regenerate entire blocks of code until it "feels" right and works correctly.
- **Parallel Feature Building**: Developers can prompt multiple AI instances (like Cursor or Windsurf) to build different features simultaneously.
